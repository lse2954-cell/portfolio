Subway Map UX Analysis

각 지하철 노선은 서로 다른 색상으로 구분되어 있으며, 환승역은 강조된 기호로 표현되어 있다. 이러한 시각적 요소들은 사용자가 별도의 학습 없이도 정보를 해석할 수 있도록 돕는다. 즉, 사용자는 단순히 색을 따라가는 것만으로도 이동 경로를 파악할 수 있어 시그니파이어가 충족되어있으며 노선도는 실제 지리적 형태를 그대로 반영하지는 않지만, 노선 간의 연결 관계와 환승 구조는 정확하게 유지한다. 이를 통해 사용자는 복잡한 도시 구조를 고려하지 않고도 이동 경로를 직관적으로 이해할 수 있다. 따라서 현실과 정보의 연결이 자연스럽다.

사용자는 노선도를 통해서 출발지에서 목적지까지의 과정을 구조화한다. 이 과정에서 형성된 개념모형은 이후 반복적인 이용 상황에서도 재사용되며, 전반적인 이용 경험을 향상시킨다. 또한 지하철 노선도는 인지 부담을 최소화하기 위해 의도적인 단순화를 적용한다. 실제 거리나 지형 정보는 제거되고, 노선은 직선과 일정한 각도로 정리되어 표현된다. 역 간 간격 역시 균일하게 조정되어 있다. 이러한 단순화는 정보의 정확성보다 이해의 용이성을 우선시한 설계로, 사용자가 빠르게 판단할 수 있도록 돕는다.

현재 위치나 환승 지점, 이동 경로 등을 시각적으로 확인할 수 있어 사용자는 자신의 이동 상태를 지속적으로 점검할 수 있다. 이는 사용 과정에서의 불안감을 줄이고, 올바른 방향으로 이동하고 있다는 확신을 제공하는 피드백이 잘 형성되었음을 보여준다..

종합하면, 지하철 노선도는 현실 정보를 그대로 재현하기보다는 사용자의 인지 과정에 맞게 재구성된 디자인이다. 이는 복잡한 도시 교통 체계를 단순하고 명확한 구조로 변환함으로써, 사용자가 빠르고 정확하게 의사결정을 내릴 수 있도록 지원한다.

UX 요소 분석

개념 설명
어포던스 노선을 따라 이동 경로를 자연스럽게 탐색하도록 유도한다.
시그니파이어 색상과 기호를 통해 노선과 환승 정보를 직관적으로 전달한다.
피드백 현재 위치와 이동 경로를 지속적으로 확인할 수 있다.
매핑 노선 간 연결 구조가 실제 이동 흐름과 직관적으로 대응된다.
